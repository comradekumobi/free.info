Hi!

The purpose of this doc is to provide a simplistic and safe catalogue of pirating sites. Pirating can often seem dangerous and complicated to people who have never done it before, so I made this guide to show that it's completely safe and accessible! This doc covers all types of media such as film, tv, games, music, books, academic resources, and even getting unlimited full versions of software such as the adobe suite completely for free.

Since these resources can be used to save A LOT of money (I have some fun statistics about that at the bottom of the doc) I highly encourage you to find independent artists whose art you enjoy and throw some money their way.

If you have any suggestions or questions please let me know! I'd love to expand the doc to meet more needs, so if there is some paid software you want and you don't see it on here, I can look around for you! Also, feel free to share this doc around! The more people cancelling their Netflix or Adobe subscriptions the better. :)

Beside each link are some tags. Tags for links in this doc include:

- Streaming: This is the simplest and safest to use. This website will stream the media you're looking for to your browser. For example, you can search for a move you want to watch on movie-web.app and it will stream it directly to you, no download required :)

- Download required: This means it requires a download to function. For example, a pdf if you're trying to get a book.

- Setup Required: This needs to be downloaded and setup separately. GenP, a program that unlocks the adobe suite for free, requires a few steps to setup, so this may take some technical knowledge. Guides for how to install these are in included in the link to it.

- Sideloading Required: This means it is an app that you cannot get on the native app stores to your mobile device. You'll need to use your pc to install a third-party app store to get the app on your mobile device. A guide for how to do this on IOS and Android is included below.

- My favorites: Self examplanitory. Just my favorite links to use.

BEFORE USING ANY OF THESE LINKS!!

Make sure to install uBlock. The installation is simple, follow the link attached and scroll to the bottom and find your internet browser of choice. Follow the instructions to install uBlock as a plugin to your browser and you're all set! uBlock is a very important tool because it blocks ads, trackers, coin miners, popups, annoying anti-blockers, and malware sites.

The final frontier of capitalism is your attention. So here are some tools that make sure all that advertising money goes to waste and all the content they've locked behind paywalls is free and available.

happy pirating!
